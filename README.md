这个项目源于我学习成人芭蕾时产生的一个困惑：为什么班里的同学明明一直在进步，但课堂的整体进度却总是推进得很慢？

后来我意识到，这可能不是某一个学生的问题，而是整个课堂作为一个系统的结果。成人课堂会不断有新学生加入，老学生会缺课，老师每天都要根据大家的掌握情况决定是否推进新的内容。课堂进度是在学生、教师和课程之间不断反馈、不断调整的过程中形成的。

于是，通过vibe coding，我尝试用 Agent-Based Modeling（基于智能体的建模）来描述这个过程。模型中，每位学生都拥有自己的基础能力、学习效率、掌握程度和出勤情况；教师则根据老学生的平均掌握度、新学生比例等信息决定下一节课是否升级课程内容。随着课程不断进行，学生会成长、遗忘、逐渐掌握“如何学习”，课堂也会呈现出不同的演化轨迹。

虽然这个模型以成人瓦冈诺娃芭蕾课堂为案例，但我真正想讨论的问题其实更加普遍：在一个不断有人加入、不断有人离开的开放课堂里，学习究竟是如何发生的？未来，我希望继续加入学习笔记、同伴互助、不同教师策略等变量，把这个模型扩展成一个可以研究成人教育、企业培训乃至开放学习社区的模拟框架。
